# qxub Examples

This directory contains practical examples demonstrating qxub features.

## Directory Structure

- `monitor/` - Examples for job monitoring and parallel submission workflows
- `parallel/` - Examples for parallel job submission patterns

## Usage

These examples are designed to be educational and can be adapted for your own workflows. Each subdirectory contains its own README with specific usage instructions.

## Note

These examples may create temporary files and submit test jobs. Review the scripts before running them in your environment.
