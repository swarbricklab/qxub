# Monitor Examples

This directory contains examples demonstrating qxub's monitoring capabilities.

## Demo Scripts

- `demo_live_monitor.sh` - Live monitoring with emoji status indicators
- `demo_summary_mode.sh` - Summary table for many jobs
- `demo_mixed_outcomes.sh` - Monitor jobs with different outcomes
- `demo_periodic_monitor.sh` - Periodic status checking
- `demo_summary_comparison.sh` - Compare live vs summary modes
- `demo_summary_live.sh` - Live summary table updates

## Quick Test Scripts

- `quick_parallel_demo.sh` - Quick parallel job submission demo
- `quick_mixed_test.sh` - Quick test with mixed job outcomes

## Job ID Files

- `demo_job_ids.txt` - Sample job IDs for demo purposes
- `quick_jobs.txt` - Job IDs for quick tests
- `quick_summary_jobs.txt` - Job IDs for summary mode tests
- `job_ids.txt` - General job ID list
- `slow_jobs.txt` - Longer-running job examples

## Usage

1. Review the scripts to understand the monitoring features
2. Adapt job IDs and parameters for your environment
3. Run scripts to see live monitoring in action

**Note**: These scripts may submit test jobs to your PBS system. Review before running.
